# 📚 Cadastro de Alunos com Tkinter e SQLite

Este projeto foi desenvolvido para praticar Python com interface gráfica usando **Tkinter** e banco de dados **SQLite**.  
A ideia é ter um sistema simples de cadastro de alunos, onde posso inserir nome e idade e visualizar todos os registros em uma tabela.

## 🚀 Funcionalidades
- Cadastrar alunos com nome e idade
- Visualizar todos os alunos cadastrados em uma tabela
- Barra de rolagem para navegar por todos os registros
- Linhas alternadas em cores para facilitar a leitura
- Ordenação ao clicar no cabeçalho da tabela

## 🛠️ Tecnologias utilizadas
- Python 3
- Tkinter (para interface gráfica)
- SQLite (para armazenamento dos dados)
